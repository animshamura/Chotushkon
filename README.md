# Chotushkon
<div align="justify">
Chotushkon, a compact project of three core services  named Gunik, Nirban and Mokabela. In extension, Mokabela has two services named Seba and Tohbil. 
  
### Gunik :
Gunik is basically a rumour scanner, developed with a model being preprocessed from a vast dataset which contains both Bangla and English text data. Gunik's front end has the functionality to check whether a news is rumour or not. If the data is already in the database, FE will call the model, it will fetch data through the backend from the database. 

### Nirban :
Nirban is basically a security app. It gets a help request and alerts the armed forces nearby. It also goes through Gunik's verification to check whether the help request is false. 

### Mokabela :
Mokabela is basically a disaster management app, which raises fund and distributes it to the victims. It has two services under named Tohbil and Seba. Tohbil raises fund and allocate resources. Seba provides the distribution with proper scrutiny. For example, to detect any fraud or false call for help, Seba has an ML model to check whether it is a fraud call. Image dataset would work there to verify.
